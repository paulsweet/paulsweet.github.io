JAVA 包和类
2021-06-29

提炼：

  • 就像文件与文件夹的区别一样。一个包中可以有多个不同的类。而同一个类在一个包中只能存在一个。Java中的一个包就是一个类库单元，包内包含有一组类，它们在单一的名称空间之下被组织在了一起。

  • 导入包名，就是要提供一个管理名称空间的机制。在Java中对名称空间进行完全控制并为每个类创建唯一的标识符组合就成为了非常重要的事情。
